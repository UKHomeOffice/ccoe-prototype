How does cloud training work in the Home Office?

The Public Cloud Team works alongside the Chief Technology Officer's (CTO) office to organise cloud training for staff. The Home Office does not provide cloud training directly. Instead, we have training offered by our cloud providers such as Amazon Web Services and Microsoft Azure and cloud training websites such as Cloudacademy.

Back to start of navigation tree

Types of training offered:

Virtual Classroom Training from AWS and Microsoft Azure

For live virtual training we have events provided by AWS and Microsoft Azure. The Public Cloud Team can coordinate ad hoc training days taught by cloud professionals. See our ‘upcoming training section’ for future training days. We can also organise cloud game days as and when required, so please contact us if this is something you are interested in. Email: public.cloud@homeoffice.gov.uk

Online learning with CloudAcademy

CloudAcademy is an online resource that can provide a learning path for AWS, Azure and Google Cloud, and we can arrange for you to view the full list of courses to help assess whether it would be suitable for you and/or your team. To arrange access to Cloud Academy, please contact us through email: public.cloud@homeoffice.gov.uk

Online learning with AWS and Microsoft Azure

AWS and Microsoft also have their own learning pathways that are free. This training is a similar style to CloudAcademy training where you can watch videos and carry out practical training/labs at your own pace.
